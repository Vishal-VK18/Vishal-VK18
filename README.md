## Featured Projects

<details>
<summary><b>ARYA — Multi-Agent Reinforcement Learning for ISR Coordination</b></summary>
<br/>

Arya-X is a multi-agent reinforcement learning system in which four specialized AI agents autonomously coordinate ISR (Intelligence, Surveillance, Reconnaissance) sensor allocation, built as a finalist submission for the Meta × PyTorch × Hugging Face OpenEnv Hackathon.

| Category | Details |
|:--|:--|
| **Stack** | Python · Qwen2.5 · GRPO · LoRA |
| **Scale** | 4 specialized autonomous AI agents |
| **Performance** | 79% reward improvement across training runs |
| **Security** | Zero hard-coded rules — fully emergent coordination logic |
| **Impact** | 50% drop in agent conflicts, 30% coordination score gain |
| **Repository** | Meta × PyTorch × Hugging Face OpenEnv Hackathon submission |

ARYA demonstrates applied multi-agent RL engineering — combining GRPO-based training with LoRA fine-tuning on Qwen2.5 to produce emergent cooperative behaviour between agents without manually scripted rules. The project reached the Top 800 of 52,000+ teams in the Meta × PyTorch × Hugging Face OpenEnv Hackathon.

</details>
